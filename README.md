## 中文

### 關於

Jack Website 是由 [Jack](https://github.com/jack08032026-hub) 開發的線上工具導航網站，專為開發者、設計師和創作者整合最實用的線上資源。

### 功能特色

- **AI 工具** - 收录 ChatGPT、DeepL、Stable Diffusion 等 AI 助手與生成模型
- **開發工具** - 整合開發必備的程式碼編輯器、版本控制、資料庫管理等工具
- **設計工具** - 收錄設計師常用的設計資源、圖片處理、色彩工具
- **生產力** - 集郵件、協同辦公、Note 工具於一處
- **學習資源** - 推薦優質的線上學習平台與教程
- **媒體工具** - 影片剪輯、音訊處理、內容創作工具
- **雲端服務** - 雲端部署、儲存、主機服務
- **資安工具** - 安全檢測、滲透測試、加密工具

### 網站特色

- 🔍 **智慧搜尋** - 支援中文、英文關鍵字快速找到所需工具
- 🌐 **雙語支持** - 完整的中文與英文界面，適合不同語言使用者
- 📱 **響應式設計** - 支援手機、平板、桌面電腦
- ⚡ **快速載入** - 輕量高效能的 Next.js 架構

### 目前統計

- 8 個工具分類
- 100+ 個精選工具

### 技術堆疊

- [Next.js](https://nextjs.org) - React 框架
- [Tailwind CSS](https://tailwindcss.com) - 樣式系統
- [TypeScript](https://www.typescriptlang.org) - 類型安全

### 如何使用

```bash
# 複製專案
git clone https://github.com/jack08032026-hub/jack-website.git

# 安裝依賴
npm install
# 或
bun install

# 啟動開發伺服器
npm run dev
# 或
bun dev
```

開啟 [http://localhost:3000](http://localhost:3000) 查看網站。

歡迎提交 Issue 或 Pull Request 建議新工具或改進網站！
